== Description ==

Add Comscore tracking code to your head section. Just activate plugin and set your C2 ID.

* Author: [Marco Canestrari](http://www.marcocanestrari.it/)

== Installation ==

1. Upload the files to the /wp-content/plugins/comscore-tag/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to "Comscore" page from Settings and save your C2 ID
